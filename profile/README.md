## ⚙️ 存量代码治理

> 《周易·系辞》曰："日新之谓盛德，生生之谓易。"

可用，不过温饱之线；好用，方入匠心之境；**久用而不衰，乃生产之道。**

代码如器，久置则锈，因循则腐。旧系统之患，非一日之寒——  
接口如积薪，层层叠叠；锁如坐牢，争而不让；内存如漏卮，注而不满。  
《大学》有言："苟日新，日日新，又日新。"  
吾辈治码，亦当如是：**不以可用为终，不以好用为止，以久用为期。**

| 层次 | 症状 | 方向 |
|------|------|------|
| 可用 | 能跑，偶尔抖 | 消除崩溃、修复边界 |
| 好用 | 响应尚可，偶有卡顿 | 降延迟、减抖动 |
| 久用 | 高负载稳定，长周期无劣化 | 内存 / 锁 / IO 深度治理、可观测性建设 |

> 《荀子》曰："不积跬步，无以至千里。"  
> 性能非一役可竟，乃日拱一卒之功。
